# Production-Ready Document Classifier

A machine learning pipeline that categorizes text documents into specific topics using Natural Language Processing (NLP). This project demonstrates model persistence, pipeline architecture, and interactive predictions.

## Tech Stack
- **Language:** Python 3
- **Libraries:** scikit-learn, joblib
- **Architecture:** `TfidfVectorizer` + `LogisticRegression` Pipeline
